Rocket API:

controls

  • controls.thrust - thrust percentage; value from 0 to 100
  • controls.gimbal - thrust angle; value from -1 (left) to 1 (right)

sensors

  • sensors.altitude - current altitude
  • sensors.position - current horizontal position (negative is left of center, vice versa)
  • sensors.angle - angle in radians (0 is up, negative is left, positive is right)

input (not available in all scenarios)

  • input.keys - boolean array of keypress states (e.g. if (input.keys[KEY.LEFT]) {...})
  • KEY - key code constants (KEY.LEFT, KEY.RIGHT, KEY.UP, KEY.DOWN)
